# T-rex Game

## 개발 환경
* Windows 11 64 Bit
* Python 3.10

## 실행 방법
* 가상 환경 생성(최초 1회)
```
python -m venv venv
```

* 가상 환경 활성화
```
.\venv\Script\activate
```

* 의존 라이브러리 설치(최초 1회)
```
pip install -r requirements.txt
```

* 메인 python 파일 실행
```
python Game7.py
```
